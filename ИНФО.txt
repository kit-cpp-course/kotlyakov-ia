Triangulation.exe определяет конфигурацию по агрументам комндной строки

Доступные ключи:
/file - файл с описанием монотоннго многоугольника (см. 123.txt для примера).
	Формат файла: перая строка - количество вершин в верхней монотонной цепочке, 
	координаты данных вершин. Вторая строка - количество вершин в нижней монотонной цепочке, 
	координаты данных вершин. Данный ключ обязателен.
/save - файл .jpg, в котором будет находиться графический результат триангуляции.
	Опциональный ключ. По умолчанию будет создано изображение "polygon.jpg".
